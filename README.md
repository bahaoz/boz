# Melontik Karlılık + Ortak Stok

## Kurulum
1. `melontik-profit-stock` klasörünü `wp-content/plugins` altına kopyalayın.
2. WordPress yönetim panelinde **Eklentiler** ekranından etkinleştirin.
3. WooCommerce etkin değilse eklenti çalışmaz.

## Kullanım (v1)
- Ürün düzenleme ekranında **Karlılık** sekmesinden alış fiyatı, ambalaj maliyeti ve sabit gider payını girin.
- Sipariş düzenleme ekranında **Karlılık Maliyetleri** kutusundan kargo maliyeti, ödeme kesintisi ve komisyon değerlerini girin.
- **Karlılık > Raporlar** ekranında tarih/kanal filtreleriyle kârlılık raporlarını inceleyin.
- **Karlılık > Ayarlar** ekranında varsayılan kargo maliyeti ve ödeme kesintisi oranını belirleyin.

## Yol Haritası
- v2: CSV ile Trendyol/Hepsiburada sipariş importu.
- v3: API ile otomatik sipariş çekme (cron).
- v4: Ortak stok senkronizasyonu.
